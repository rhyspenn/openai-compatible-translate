# 更新日志

## [2.0.0] - 2024-12-19

### 🎉 重大更新
- **全新架构**：从智谱 AI 专用插件升级为通用 OpenAI 兼容插件
- **通用兼容**：支持任何 OpenAI API 兼容的服务

### ✨ 新增功能
- 支持自定义 API Base URL
- 支持自定义模型名称，默认使用 `gpt-4.1`
- 新增 Temperature 参数配置
- 新增 Max Tokens 参数配置
- 扩展语言支持至 35+ 种语言

### 🔧 改进
- 优化 API 调用逻辑
- 改进错误处理和用户提示
- 更新文档，添加多服务配置示例

### 📝 兼容性
- 支持 OpenAI 官方 API
- 支持 Azure OpenAI Service
- 支持通义千问、智谱 AI 等国内服务
- 支持本地大模型（通过 LM Studio、Ollama 等）
